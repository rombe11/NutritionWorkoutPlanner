Nutrition & Workout Planner

A Windows desktop application built with C# WinForms that generates personalized nutrition menus and workout plans based on your BMR (Basal Metabolic Rate) and food preferences.

Features

Calculate your BMR based on age, weight, height, and activity level.

Generate customized meal plans according to your preferred foods.

Create workout plans tailored to your fitness goals.

Save or print your plans for daily use.

User-friendly Windows interface.

Requirements

Windows 10/11

.NET Runtime

The application is packaged as a .exe, so no development environment is needed.
